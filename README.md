**musync** — это продвинутый локальный музыкальный плеер и менеджер библиотеки, созданный для тех, кто ценит полный контроль над своей коллекцией, надежность хранения данных и эстетику.

## ✨ Ключевые возможности

### 🎶 Продвинутое управление библиотекой

* **Авто-сканирование и тегирование:** Чтение ID3-тегов (через `mutagen`), нормализация имен исполнителей и безопасное переименование файлов.
* **Умный поиск дубликатов:** Группировка треков на основе нормализованных названий и метаданных.
* **Кастомизация обложек:** Извлечение доминирующего цвета (с защитой от ч/б обложек) и генерация квадратных миниатюр одинакового размера для красивого отображения.

### 📥 Синхронизация с YouTube

* **Интеграция с `yt-dlp`:** Загрузка отдельных треков и целых плейлистов напрямую с YouTube.
* **Защита от дубликатов и мультипоточность:** Быстрое скачивание без риска засорить библиотеку повторами.

### 🎤 Синхронизированные тексты песен

* Поддержка файлов `.lrc` и интеграция с внешними API (**Better Lyrics**, **LRCLib**).
* Построчная и пословная синхронизация текстов с воспроизведением.

### 📊 Глубокая статистика (Ваш личный Spotify Wrapped)

* Отслеживание истории прослушиваний.
* Расчет серий (стриков).
* Построение тепловой карты прослушиваний по часам.
* Топы треков и исполнителей за неделю, месяц и год.

### 💾 Надежность "под капотом"

* **Атомарное сохранение:** Использование `.tmp` файлов и потокобезопасных блокировок (`RLock`), гарантирующих, что ваша база данных (JSON) никогда не повредится при сбоях или внезапном закрытии приложения.

## 🚀 Установка

1. Склонируйте репозиторий:
   
   ```bash
   git clone https://github.com/WhereGamer/Musync.git
   cd Musync
   ```

2. Установите необходимые зависимости:
   
   ```bash
   pip install -r requirements.txt
   ```
   
   *(Убедитесь, что у вас установлен FFmpeg, так как он требуется для работы yt-dlp и обработки аудио).*

## 🛠 Запуск

Запустите основной файл проекта:

```bash
python main.py
```

## ⚙️ Конфигурация

При первом запуске `musync` автоматически создаст файл конфигурации и необходимые директории:

* `music/` — директория для ваших аудиофайлов.
* `cache/` — кэш приложения (обложки, тексты).
* `config.json` — настройки путей и параметров плеера.
